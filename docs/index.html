This software is an implementation of a group-key agreement protocol for ad hoc networks.

Features

AHGKA

This software is an implementation of a GKA system derived from the one described in "AGDH (Asymmetric Group Diffie Hellman) An Efficient and Dynamic Group Key Agreement Protocol for Ad Hoc Networks" from R. Bhaskar, D. Augot, C. Adjih, P. Muhlethaler, S. Boudjit, (see there and also INRIA RR-5915)

Network protocol has been enhanced (documented in an internal report), and the implementation was mostly written by Rajesh Kumar.
